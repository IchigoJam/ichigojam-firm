# IchigoJam

## for SPRESENSE

### 準備

- Arduino SPRESENSE 開発環境 (path /Users/$USER/Library/Arduino15/packages/SPRESENSE/ 想定)

### build

```sh
cd build-SPRESENSE
sh build.sh
```

## write

```sh
cd build-SPRESENSE
sh write.sh
```
